| Hotkey   | Action                                              |
| -------- | --------------------------------------------------- |
| ⌘ 9      | Fit                                                 |
| ⌘ 0      | Actual Size                                         |
| ⌥ ⌘ 1    | Hide Sidebar                                        |
| ⌥ ⌘ 2    | Show Thumbnails                                     |
| ⌥ ⌘ 5    | Show Bookmarks                                      |
| ⌥ ⌘ G    | Go to Page...                                       |
| ⌘ D      | Add Bookmark                                        |
| ⌘ 1      | Continuous Scroll                                   |
| ⌘ 2      | Single Page                                         |
| ⌘ 3      | Two Pages                                           |
| ←/→, ↑/↓ | (if zoomed) Move Area (if fit) Previous / Next page |
| ⌥ ←/→    | Move page back or forward                           |
